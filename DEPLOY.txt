پروژه با لاراول ورژن 11.41.0 و php ورژن 8.2.13 پیاده سازی شده است.
تمام dependency ها با composer نصب شده است که با composer install قایل نصب هستند.
دیتابیس پروژه mysql میباشد.
صف ها از کانکشن sync استفاده میکنند ولی میتوان به سادگی با استفاده از سوپروایزر و کانکشن دیتابیس و غیره به شکل async اجرا کرد.
دستور مناسب برای سوپروایزر دستور زیر می باشد که میتوانید پارامتر ها را با توجه به نیاز و منابع سرور تغییر دهید
php artisan queue:work --sleep=3 --max-jobs=10 --tries=3 --max-time=3600
ارسال کننده ایمیل (MAIL_MAILER) برای تست لاگ هست و ایمیل های ارسالی در فایل لاگ قابل مشاهده هستند که کانفیگ مربوط به میتوان از فایل env تغییر داد
دیتابیس پروژه در حالت تست sqlite میباشد که در حالت in memory اجرا میشود.
برای مدیریت جاب ها باید کرونجاب زیر در سرور تعریف شود:
* * * * * cd /path-to-your-project && php artisan schedule:run >> /dev/null 2>&1
زمان اعتبار کد ارسال شده به ایمیل کاربران برای اعتبارسنجی به طور پیشفرض 2 دقیقه میباشد که میتوانید با استفاده از کلید VERIFY_CODE_LIFE_TIME در فایل .env مقدار آن را بر حسب ثانیه تغییر دهید.
اعتبار توکن بعد از لاگین به طور پیشفرض 15 دقیقه میباشد که میتوانید با استفاده از کلید TOKEN_LIFE_TIME در فایل .env مقدار آن را برحسب دقیقه تغیر دهید.
فایل env.testing فقط برای تست میباشد
ایجاد فایل داکیومنت با دستور زیر
php phpDocumentor.phar run -d . -t docs/api
